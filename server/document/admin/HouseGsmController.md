- [HouseGsmIndexRequest](../OBJECT.md#HouseGsmIndexRequest)

Параметры: 

- [HouseGsmStoreRequest](../OBJECT.md#HouseGsmStoreRequest)

Параметры: 

- `int` *id* Идентификатор абонента в GSM
